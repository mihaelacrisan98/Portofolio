Basket în Brașov

Revamping Brașov's basketball scene was my game with 'Hoops Renovation,' part of Generația Tech's drive! From August to October 2023, we built a web hub for locals, tracking basketball court makeovers. I spiced up the sign-in page for smooth sailing. My HTML and CSS flair made account creation a breeze! Guess-my-number

This project is a simple number-guessing game where a player needs to guess a randomly generated secret number. The game provides feedback on whether the guessed number is too high, too low, or correct.

Pig-Game

This project is a simple web-based dice game where two players take turns rolling a dice and accumulating points. The game consists of a web page with two player panels, each displaying their current score and total score. The page also includes buttons for rolling the dice, holding the current score, and starting a new game.
